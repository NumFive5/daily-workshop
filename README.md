# 日常工作坊 (Daily Workshop)

日常工作坊 - 前端项目汇总与实验场。

## 🚀 项目介绍
本项目是一个基于 **Flutter Web** 构建的个人工作坊平台，集成了多种日常管理与展示模块。原为考勤管理系统，现已全面升级为通用工作坊架构。

## 🛠️ 核心功能
- **工作坊管理**：提供数据上传、处理与展示的综合工具。
- **时间轴 (Timeline)**：记录工作坊的重要里程碑与日常点滴。
- **相册 (Gallery)**：展示项目截图、设计稿及生活瞬间。
- **诗歌 (Poem)**：内置人文艺术展示模块。
- **动态导航**：支持 Material Design 3 的响应式导航体验。

## 💻 技术栈
- **Frontend**: Flutter 3.x (Web)
- **Styling**: Material 3
- **Deployment**: Docker / Python HTTP Server

## 📦 部署指南
1. 克隆仓库：`git clone https://github.com/NumFive5/daily-workshop.git`
2. 运行本地预览：
   ```bash
   cd build/web
   python3 -m http.server 8000
   ```
